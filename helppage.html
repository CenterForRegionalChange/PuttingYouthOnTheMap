Frequently Asked Questions

Why aren't my pop-up boxes appearing?

You will need to disable your pop-up blocker in your browswer in order to have full fuctionality of this site. Every browser has a slightly different way to do disable pop-up blockers, but you might see an alert in your URL bar when a pop-up has been blocked. Click on it and always allow pop-ups for this site. Or add interact.regionalchange.ucdavis.edu as an exception to your pop-up settings. You should only need to do this once for each browser.

When I change data years I sometimes don't see any data on the map. Is this an error?

This is not an error. Currently, we list all years for which we have any data (2010, 2011, 2012, and 2013), but not all layers have data for all of these years. If we have data for a paticular year it will be mapped; if not, then a blank map will appear.
